# Changelog

## v0.1.0-alpha

Initial public alpha.

- Wayland layer-shell overlay window.
- Clickable setup mode for login, cookies, and positioning.
- Click-through overlay mode for playing on a single monitor.
- Kick, Twitch, YouTube, and custom URL presets.
- Width, height, corner, margins, opacity, and nudge controls.
- Experimental safe hiding of the chat message input.
- CachyOS/KDE Plasma Wayland install script.
